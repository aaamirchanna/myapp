Python

Python is a free open source programing langugage which can be used for multiple programs. Python can solve most of your problems. It is used for web developeent(server), desktop applications data analysis and many more. Python can be used on different platforms like Windows, Linnux, IOS and Andriod.

Python is strong typing languague. It does not need complier. Python's community developed a pytnon package which create an environment to run Python on Windows. Python is very easy to learn. Its syntax is very simple you need to learn basic English. Any one can code who knows simple language. A small program can solve lot of problems.

For Windows you need yo install python from www.python.org.

Python can be on any of following IDEs

- Pycharm

- Visual Studio Code

- Atom

- IDLE

There are following main advantages of Python

- Easy to learn, read and code

- Free and open source

- Thousands of Libraries support

- Don't need any interpretor

- Productive

- Easily Portable

Disadvantages of using Python

- It is very time consuming. Ithas has a low execution speed

- Python is not good for Mobile application

- Incompatibility of two versions

- Require additional testings

- Depends on framework and liabraries

Python is best for new developers. Basic English knowledge is needed. Python is magic language for newbies.
