What is Javascript

Javascript was in developed when the Java was mst popular languague. Today Javascript itself is most popular language in the world. It is belevided that there is no alternative to it. It was invented by Brendan Eich in 1995. It was handed over by netscape in 1997 to ECMA. ECMA is developing its new releases. Currently ES6 is used which is released in 2015.

Initially it was created for web developemnt but now it is widely used langugage. Big tech companies like Google, Facebook Netflix Walmart, youtube, and Paypal etc are using Javascrit. The developers comunity and investment companies are supporting developing Javascript as all best rounder. Developers comunity has embedded the Google chrome open source engine with c++ and named it Node js. Next js is same developing phase of Node js.
